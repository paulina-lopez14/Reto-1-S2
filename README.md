## Simulador de farmacia con descuento
🏥 Sistema Básico de Entradas en Java 🏥

## 📌 Intro 
Este programa permite calcular el total de una compra en farmacia y aplicar un descuento si supera cierta cantidad. Este reto refuerza el uso de operadores aritméticos, asignación, entrada por consola, e inferencia de tipo con var.

## ✨ Instrucciones
💊1. Crea una clase llamada SimuladorFarmacia.

💻 2. Solicita al usuario que introduzca los siguientes datos:

Nombre del medicamento (tipo String)
Precio unitario (tipo double)
Cantidad de piezas (tipo int)
Calcula el total sin descuento multiplicando el precio por la cantidad.

🛍️3. Supón que si el total supera los $500.00, la farmacia ofrece un 15% de descuento. Aplica el descuento usando el operador ternario (? :) sin estructuras if.

😎4. Muestra en consola el resumen con el siguiente formato:
Medicamento: Amoxicilina
Cantidad: 5
Precio unitario: $120.0
Total sin descuento: $600.0
¿Aplica descuento?: true
Descuento: $90.0
Total a pagar: $510.0

## 📂 Estructura 
- Simuladorfarmacia.java: Clase principal que contiene el main, solicita los datos de la compra con scanner (nombre del medicamento, precio y cantidad), calcula el total, determina si aplica el 15% de descuento con el operador ternario y muestra el resumen de la compra.
  *Nota: para ocupar el scanner en el archivo Principal.java, se pone lo siguiente al inicio del archivo "import java.util.Scanner;"*
